# 📍 Location Intelligence Dashboard

A React-based Location Intelligence Dashboard that allows users to search locations, explore nearby places, and visualize results on an interactive map using Google Maps, Places, and Directions APIs.

## 🚀 Features

- Search any location worldwide  
- Discover nearby places like restaurants, cafes, hotels, hospitals  
- Interactive Google Maps integration  
- Place markers with details  
- Directions & route visualization  
- Fast and responsive UI using React  

## 🛠️ Tech Stack

- React.js (Vite)
- JavaScript (ES6+)
- Node.js & Express (if used)
- Google Maps JavaScript API
- Google Places API
- Google Directions API
- CSS / Tailwind

## ⚙️ Setup Instructions

### Clone the repository
git clone https://github.com/sandhiyyaa-v/location-intelligence-dashboard.git

### Go to project folder
cd location-intelligence-dashboard/client

### Install dependencies
npm install

### Run the project
npm run dev

## 🔑 Environment Variables

Create a .env file in client/:

VITE_GOOGLE_MAPS_API_KEY=your_api_key_here
